# Vocabulário banido (AI slop)

LLMs treinadas em texto corporativo em inglês traduzem essas palavras
automaticamente. São o cheiro mais forte de AI slop. Substitua sempre.

## Lista de busca-substitui

| Banido | Substitua por |
|---|---|
| delve / aprofundar-se em | examinar, explorar, estudar |
| crucial | importante, essencial |
| robust | sólido, completo, resiliente |
| comprehensive | completo, abrangente |
| nuanced | matizado, com camadas |
| multifaceted | com múltiplas dimensões |
| furthermore / moreover / additionally | além disso, também, ainda |
| pivotal | decisivo, determinante |
| landscape (de mercado) | cenário, panorama |
| tapestry | tecido, conjunto |
| underscore | reforçar, sublinhar |
| foster | cultivar, promover |
| showcase | mostrar, exibir |
| intricate | detalhado, complexo |
| vibrant | vivo, intenso |
| significant | relevante, expressivo |
| seamless | sem fricção, contínuo |
| empower | dar poder, capacitar |
| leverage (verbo) | usar, aproveitar |
| streamline | simplificar |
| cutting-edge / state-of-the-art | mais recente, de ponta |
| game-changer | divisor de águas (com moderação) |
| paradigm shift | mudança de modelo |

## Frases de abertura banidas

- "É importante notar que..." → corte e escreva direto.
- "No mundo de hoje..." → corte.
- "Em uma era em que..." → corte.
- "Vivemos em tempos onde..." → corte.

## Frases de fechamento banidas

- "Em conclusão..." → conclua sem anunciar.
- "Em suma..." → corte.
- "Para concluir..." → corte.
- "Espero que este artigo tenha sido útil." → corte.

## Conectivos vazios banidos

- "É válido ressaltar que..."
- "Vale lembrar que..."
- "Como já foi dito anteriormente..."
- "Não obstante..."

## Validação

A skill `conteudo` faz grep por essas palavras antes de salvar. Se encontrar,
para e pede revisão.
