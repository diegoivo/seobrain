# Voz PT-BR

LLMs frequentemente derrapam para PT-PT, formalidade burocrática, ou tradução
literal do inglês. Estas são as regras hard-checked pela skill `conteudo`.

## PT-BR estrito (não PT-PT)

| Errado (PT-PT) | Correto (PT-BR) |
|---|---|
| ecrã | tela |
| ficheiro | arquivo |
| telemóvel | celular |
| equipa | equipe / time |
| autocarro | ônibus |
| pequeno-almoço | café da manhã |
| utilizador | usuário |
| comboio | trem |
| eléctrico | elétrico |
| facto | fato |
| acção | ação |
| director | diretor |
| ligação (chamada) | chamada |
| sítio (web) | site |

## Sem gerundismo

| Errado | Correto |
|---|---|
| vou estar enviando | vou enviar / envio |
| vamos estar fazendo | vamos fazer / faremos |
| vou estar conectando você | vou conectar você |

## Sem voz passiva burocrática

| Errado | Correto |
|---|---|
| foi feito por nós | fizemos |
| será analisado | analisaremos |
| recomenda-se que | recomendamos |
| pode ser observado | observe |

## Pessoa: você (não tu, não vós, não a gente)

- ✅ "você precisa", "você pode escolher"
- ❌ "tu precisas", "vós precisais"
- ⚠️ "a gente" — só em copy bem coloquial; em texto editorial, use "nós"

## Densidade

- Frases curtas (12-18 palavras médias).
- Um verbo principal por frase.
- Sem "muito", "bastante", "extremamente" — use o adjetivo forte direto.

## Formalidade calibrada

- Editorial: você, segunda do plural não. Tom de "colega que sabe do assunto".
- Não chame o leitor de "amigo", "pessoal", "galera".
- Não use "caros leitores", "querido leitor".

## Anglicismos

Mantém em inglês quando consagrado no nicho:
- Tech: SEO, SERP, crawler, backlink, framework, deploy, build, prod, staging, branch, PR, commit.
- Marketing: funil (não funnel), conversão (não conversion), atribuição (não attribution).

Quando há boa palavra em PT-BR e o anglicismo não é consagrado, prefira PT-BR.
